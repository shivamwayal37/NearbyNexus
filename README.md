## Setup Instructions
1. **Backend**:
   - Install Java 21, Maven, MySQL, MongoDB, Redis.
   - Update `application.properties` with database credentials.
   - Run: `mvn spring-boot:run`
2. **Frontend**:
   - Install Node.js, run `npm install` in `frontend`.
   - Start: `npm run dev`
